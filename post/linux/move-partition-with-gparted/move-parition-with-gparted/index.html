Comment déplacer Une Partition Avec Gparted

Déplacer une partition avec GParted

GParted Live est une distribution Linux spécialisée dans la gestion des partitions de disque dur. Ce guide explique comment déplacer une partition d’un disque à l’aide de GParted Live.

Prérequis

Avant de commencer, disposer des éléments suivants :

  • Un iso du système GParted (https://gparted.org/downloads) :
  • Accès à un ordinateur avec le disque que vous souhaitez modifier.

L’idée ici c’est qu’on à une partition utilisée entre la partition que l’on veut étendre et la partition non alloué

Étapes

  1. Démarrage depuis GParted Live :
    • Insérez la clé USB ou l’iso contenant GParted Live dans la machine désirée. (Pour créer une clé USB bootable, se référée à la documentation pour créer une clé avec Rufus USB)
    • Redémarrez l’ordinateur et accédez au menu de démarrage (généralement en appuyant sur une touche spécifique comme F12 ou Suppr pendant le démarrage, pour les VM avec VMware, souvent la touche F2).
    • Sélectionnez la clé USB ou le CD comme périphérique de démarrage.
  2. Ouvrir GParted :
    • Une fois démarré, on sélectionne GParted Live (Default settings) qui devrait convenir dans la majorité des cas
    • Une fois démarré, dans Configuring console-data , on sélectionne Select keymap from arch list puis le type de clavier connecté (Ici azerty), puis la région (ici French) et enfin la variante du clavier, la majorité du temps PC keyboard , à noter cependant que le support du clavier Apple est possible.
    • On sélectionne ensuite le keymap a installer (ici France donc 08 ) et ensuite on fait 0 pour démarrer l’interface graphique
  3. Sélectionner la partition à déplacer :
    • Dans GParted, apparait alors une liste des disques disponibles sur votre ordinateur.
    • On cherche le disque contenant la partition que vous souhaitez déplacer (ici nvme0w1 ), pour sélectionner un disque, une liste déroulante est positionnée en haut /dev/disk (Taille Gio)
    • Cliquez avec le bouton droit de la souris sur la ou les partitions qui se trouve entre la partition que vous voulez agrandir et la partition non allouée et sélectionnez “Déplacer/Redimensionner”.
  4. Déplacer la partition :
    • Utilisez la souris pour déplacer la partition vers l’emplacement souhaité sur le disque.
    • Si l’idée est d’utiliser tout l’espace disponible, on déplace la partition de la gauche vers la droite (tout au bout) et on clique sur Redimentionner/Déplacer
    • On peut également saisir manuellement les valeurs de déplacement si souhaité.
    • Il est possible qu’un message précisant qu’il est possible que le déplacement de la partition peut empêcher le système de d’initialiser, on clique sur OK
  5. Appliquer les modifications :
    • Une fois que la partition ait été déplacée à l’emplacement désiré, on clique sur “Appliquer” pour valider les modifications (le bouton pour appliquer est un petit check vert)
    • GParted commencera à déplacer la partition. Cela peut prendre un certain temps en fonction de la taille de la partition et de la vitesse du disque.
  6. Attendre la fin du processus :
    • Une fois que GParted a terminé de déplacer la partition, un message de confirmation apparait, on clique sur Fermer / Close.
    • On redémarre la machine pour que les modifications prennent effet avec le bouton EXIT présent sur le bureau, on sélectionne redémarrer, puis on appuie sur entrée

Conseils supplémentaires

  • Assurez-vous de sauvegarder toutes vos données importantes avant de modifier les partitions.
  • Soyez prudent lors du déplacement des partitions, car des erreurs peuvent entraîner la perte de données.
