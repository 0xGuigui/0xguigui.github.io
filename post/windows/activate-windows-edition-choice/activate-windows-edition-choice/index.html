Activer le choix des éditions dans l'installeur de Windows avec ei.cfg - Tue, Sep 17, 2024

Comment transférer un driver d'imprimante vers un autre PC Windows

Introduction

Le fichier ei.cfg est utilisé pour spécifier quelle édition de Windows doit être installée lors du processus d’installation. En modifiant ce fichier, on peut activer le choix des éditions disponibles lors de l’installation de Windows. Compatible avec Windows 10 /

Configuration de ei.cfg

On crée un fichier nommé ei.cfg avec le contenu suivant :

[EditionID]
Professional
[Channel]
_Default
[VL]
0

Dans ce fichier, on peut spécifier l’édition de Windows que l’on souhaite installer en modifiant la valeur après EditionID.

Dans l’exemple ci-dessus, l’édition spécifiée est Professional. On peut modifier cette valeur pour installer une autre édition de Windows.

Sélection Automatique de l’Édition

Pour activer la sélection automatique de l’édition lors de l’installation de Windows, on peut utiliser le fichier ei.cfg suivant :

[EditionID]
Professional
[Channel]
_Default
[VL]
0

Éditions Disponibles pour la Sélection Automatique

Voici les éditions de Windows que vous pouvez spécifier dans le fichier ei.cfg pour la sélection automatique :

  • Home
  • Professional
  • Education
  • Enterprise
