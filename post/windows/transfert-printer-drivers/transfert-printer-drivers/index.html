Comment transférer un driver d'imprimante vers un autre PC Windows

Introduction

L’outil PrintBrmUi.exe sur Windows 10 permet de sauvegarder et restaurer facilement les pilotes d’imprimante ainsi que les files d’attente. Il est disponible pour les versions Pro et supérieures de Windows.

Prérequis

  • Windows 10 Pro ou version supérieure.
  • Droits d’administrateur pour certaines étapes.

Sauvegarder les Pilotes d’Imprimante et les Files d’Attente

Étapes 1

  1. Lancer l’outil de migration d’imprimante :
    • Ouvrir le menu Démarrer.
    • Taper PrintBrmUi.exe et appuyer sur Entrée.
  2. Exporter les imprimantes et files d’attente :
    • Une fenêtre « Getting started with Printer Migration » s’ouvre.
    • Sélectionner Exporter des files d’attente d’imprimantes et des pilotes dans un fichier.
    • Cliquer sur Suivant.
  3. Enregistrer le fichier de sauvegarde :
    • Choisir les imprimantes et files d’attente à sauvegarder.
    • Enregistrer le fichier dans un endroit sécurisé (disque externe ou serveur, par exemple).
  4. Finaliser la sauvegarde :
    • Le fichier .printerExport est généré. Conserver ce fichier pour une future restauration.

Restaurer les Pilotes d’Imprimante et les Files d’Attente

Étapes 2

  1. Exécuter l’outil en mode administrateur :
    • Ouvrir le menu Démarrer, taper PrintBrmUi.exe, puis exécuter en tant qu’administrateur.
  2. Importer les données sauvegardées :
    • Sélectionner Importer des files d’attente d’imprimantes et des pilotes depuis un fichier.
    • Choisir le fichier .printerExport précédemment sauvegardé.
  3. Finaliser l’importation :
    • Suivre les instructions pour terminer. Les imprimantes et files d’attente sont alors restaurés.

Notes Importantes

  • Pour Windows 10 Home, l’outil PrintBrmUi.exe n’est pas inclus par défaut. Il est possible de copier l’outil depuis un PC Windows 10 Pro vers C:\Windows\System32\ pour l’utiliser.
  • Lors de la réimportation, lancer l’outil avec les droits d’administrateur est indispensable pour garantir que tout soit restauré correctement.
